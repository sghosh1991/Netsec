1. source_code folder contains the source code of SSH, FTP, DOVECOT, JOOMLA and  WORDPRESS. Also contains source_code/README.txt file for running each of the above services.


2. LogstashConfigFiles folder contains 
   * LogstashConfigFiles/LogstashServer.conf - the configuration scripts for parsing.
   * LogstashConfigFiles/LogstashShipper.conf - configuration file for sending log messages to central server.


3. Detailed_Installation.pdf has the step by step instructions of what we have done for each of the services SSH, FTP, DOVECOT, JOOMLA and  WORDPRESS.

4. pythonscripts directory has two scripts which we used for questions 8 and 9.


4. Project4_Report.pdf contains the project report.
